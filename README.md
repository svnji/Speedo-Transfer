# 💸 Speedo Transfer

A modern money transfer iOS application built with **Swift** that simulates a secure digital wallet experience. The application allows users to authenticate, manage their balance, and transfer money through a clean and intuitive interface.

## ✨ Features

* 🔐 Secure User Authentication
* 💰 Account Balance
* 💸 Money Transfer
* 👥 Recipient Management
* 📜 Transaction History
* 🌈 Modern Gradient UI
* ⚡ Smooth Navigation Flow
* 🌐 REST API Integration

---

## 🛠 Tech Stack

* Swift
* UIKit
* MVVM Architecture
* URLSession
* CocoaPods
* REST APIs
* Codable
* Auto Layout

---

## 🏗 Architecture

The project follows the **MVVM (Model-View-ViewModel)** architecture to separate business logic from the user interface, making the codebase easier to maintain and extend.

```text
Views
   ↓
ViewModels
   ↓
Services
   ↓
Networking
   ↓
REST API
```
